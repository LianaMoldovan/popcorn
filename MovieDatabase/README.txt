Pop Corn

 • Android supported versions 21+ and targets Android phones
 
 • Picture loading - uses Glide4: https://github.com/bumptech/glide
 • REST requests Retrofit2 (app is quite small and no need to integrate rxJava or something more complex)
	 APIs:  • Search API: https://api.themoviedb.org/3/search/movie
		• Movie poster (size: w92, w185): https://image.tmdb.org/t/p
 • Custom family-fonts Lato (https://fonts.google.com/specimen/Lato)
 
 • The application uses a basic MVP architecture. This facilitate unit testing.
   Unit Tests using JUnit and Mockito
   
  Time spent ~24 hours
